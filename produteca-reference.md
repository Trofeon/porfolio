# Referencia para Clonar Produteca (Replicación Completa)

Este documento contiene la estructura completa, diseño y especificaciones de la página web *produtecalab.com.br* para ser replicada utilizando el agente `landing-page-architect`.

## 1. Diseño y Estética (Design System)
- **Colores Principales:** 
  - Azul Marino (Navy) intenso para fondos oscuros y contrastes (ej. `#0A192F` o similar).
  - Verde Cerceta/Cian (Teal/Cyan) como color de acento principal.
- **Tipografía:** `Montserrat` (o similar, sans-serif moderna, geométrica y limpia) en toda la web.
- **Botones:** Estilo "Ghost" (contorno grueso, sin relleno, fondo transparente) con bordes cuadrados o "sharp-edges". Al hacer hover, cambian a fondo sólido o invertido.
- **Estructura visual:** Diseño plano (flat design), estructurado en bloques anchos alternando fondos blancos/claros y oscuros (Azul Marino).
