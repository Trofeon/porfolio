Siempre que se solicite realizar tareas de diseño de interfaces web, UI/UX, creación de componentes, landing pages, o similares en este proyecto:

1. **Revisión Obligatoria:** Debes revisar las instrucciones documentadas en la skill de UI/UX Pro Max antes de generar cualquier código.
   - Usa la herramienta `view_file` para leer `c:\Users\Iván\Documents\GitHub\porfolio\.agent\skills\ui-ux-pro-max\SKILL.md`.

2. **Generación de Design System:** Como lo estipula la skill, antes de implementar el código, debes usar su motor de razonamiento para definir un "Design System" a medida (colocando colores, tipografías, y mejores prácticas para el caso de uso específico).

3. **Aplicación Práctica:** Aplica esos criterios directamente en el código del componente asegurando siempre un diseño *premium*, animaciones sutiles e interacciones responsivas.
